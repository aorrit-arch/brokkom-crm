    Equip
    Gestió de mediadors · només administradors
    Com convidar algú:
    1. Crea l'entrada amb el botó "+ Afegir mediador" (queda com a pendent, sense compte real encara).
    2. Vés a Supabase → Authentication → Users → Add user → invita l'email.
    3. Quan l'usuari fa servir l'enllaç i defineix contrasenya, el sistema vincula automàticament el seu compte amb el mediador (per coincidència d'email).
    A futur, el botó farà aquest pas 2 automàticament via Edge Function.
